## ✅ INTERFACE AGORA TEM:

### **1. Box Destacado (Amarelo/Laranja)**
- Impossível não ver
- Cada botão tem o seu

### **2. Três Seções Organizadas:**

**📝 MENSAGEM** (obrigatória)
- Textarea grande
- Placeholder de exemplo
- Explicação do que é

**🎬 MÍDIA** (opcional)
- Campo para URL
- Seletor de tipo (vídeo/foto)
- Aviso sobre canal público

**💰 VALORES** (obrigatórios)
- Preço adicional
- Descrição do bônus
- Explicações em cada campo

### **3. Dica no Final**
- Lembrete sobre usar mídia
- Incentiva melhores conversões
